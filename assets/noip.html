No-IP Settings (DDNS)

How it works

No-IP service binds a fixed hostname to a dynamic public IP address. This allows server access via a fixed link like mygateway.ddns.net even if the IP changes.

  • Create a free account on noip.com
  • Create a Hostname from the No-IP control panel
  • Enter your account details here and press Save
  • Press Update Now to sync DNS with the server's current IP
